# TweetMark 隐私说明 / Privacy Policy

**中文**

TweetMark 是一个完全本地运行的浏览器插件:

- 你收藏的推文内容(文字、作者、链接、图片地址)和你填写的备注,**只保存在你自己浏览器的本地存储**(`chrome.storage.local`)中。
- 插件**不收集、不上传、不发送任何数据**到任何服务器——它没有后端,代码中不存在任何网络上报逻辑,欢迎审查源码验证。
- 插件仅在 `twitter.com` / `x.com` 页面上运行,用途仅限于:检测你的收藏动作、读取被收藏推文的内容用于本地保存和分类。
- 导出功能生成的 JSON 文件保存在你自己的电脑上,由你自己保管。
- 卸载插件即删除全部本地数据(建议先导出备份)。

**English**

TweetMark runs entirely locally:

- Bookmarked tweet content (text, author, link, media URLs) and your notes are stored **only in your browser's local storage** (`chrome.storage.local`).
- The extension **collects, uploads, and transmits nothing**. There is no backend and no analytics — the source code is open for inspection.
- It runs only on `twitter.com` / `x.com`, solely to detect your bookmark actions and read the bookmarked tweet for local saving and categorization.
- Exported JSON files stay on your own machine.
- Uninstalling the extension deletes all local data (export a backup first).
